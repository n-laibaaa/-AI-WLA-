# Wumpus Logic Agent

A Knowledge-Based Agent that navigates a Wumpus World grid using Propositional Logic and Resolution Refutation.

## Setup

### 1. Backend
```
cd backend
pip install -r requirements.txt
python app.py
```

### 2. Frontend
Open `frontend/index.html` with VS Code **Live Server** extension.

## How to Play
- Click **New Game** to start
- Cells with a glowing purple border are KB-proven safe — click to move
- Watch percepts, inference steps, and the KB log update in real time
